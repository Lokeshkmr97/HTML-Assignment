DOCTYPE html: This declaration specifies the document type and version of HTML being used (HTML5 in this case).

html: The opening tag for the HTML document.
head: This section contains meta-information about the document, such as the title of the web page, which is displayed in the browser's title bar or tab.
title: The title of the web page, which will appear in the browser's title bar or tab. You can change this to whatever you like.
body: This section contains the visible content of the web page.
